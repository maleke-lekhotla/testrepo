# Budgeting App

A React budgeting app that supports:

- Category creation with required classes:
  - Fixed Costs
  - Saving and Investments
  - Credit
  - Discretionary Spending
- Uploading a screenshot of transactions.
- Categorizing transactions against custom categories.
- A chart-first dashboard that shows spending as a percentage of income, split by spending class and by planned vs actual.
- Summary cards for total planned spending as a percentage of planned income and total actual spending as a percentage of actual income.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
